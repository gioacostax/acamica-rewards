## Getting Started

```bash
# Get the latest source
git clone https://github.com/gioacostax/acamica-rewards.git <myproject>

# Change directory
cd <myproject>

# Install dependencies
yarn

# Start development server
yarn start
```

## Build

```bash
# Build your project
yarn build

# (Optional) Upload your app to Vercel
yarn deploy
```
